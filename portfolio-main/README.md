This is a premium single-page portfolio built with React + Vite.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) with your browser to see the result.

You can start editing the page by modifying `src/App.tsx`. The page auto-updates as you edit the file.

The design system includes a dark-first premium glassmorphism theme, animated section reveals, and a Three.js dotted background surface.

## Build

```bash
npm run build
npm run preview
```
